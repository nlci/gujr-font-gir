## Current version

The latest released version of the fonts is always available from the [Gir release page](https://github.com/nlci/gujr-font-gir/releases) as a .zip archive for all major platforms.

In addition to released versions, the most recent [development build](https://github.com/nlci/gujr-font-gir/releases/tag/dev) is also on the release page.

## Previous versions

Previous versions remain available from the [release page](https://github.com/nlci/gujr-font-gir/releases).

A detailed list of changes for each version of the fonts is on the [Version history page](history.md).
