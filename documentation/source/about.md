## About this project

NLCI Gujarati is one font family for the Gujarati script - Gir.

To use the prefered form of the CANDRA E vowels please see the [FAQ](faq.md).

## Future plans

Please send us your requests using [GitHub issues](https://github.com/nlci/gujr-font-gir/issues).
If you are interested in helping us make the fonts better see the [Developer](developer.md) page.
