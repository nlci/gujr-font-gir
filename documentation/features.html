NLCI Gujarati - Font Features

The NLCI Gujarati fonts have some optional features that may be useful or required for particular uses or languages. This document lists all the available features.

These OpenType features are primarily specified using four-letter tags (e.g. 'ss20'). For more information on how to access OpenType features in specific environments and applications, see Using Font Features.

This page uses web fonts (WOFF2) to demonstrate font features and should display correctly in all modern browsers. For a more concise example of how to use the NLCI Gujarati fonts as a web font see Gir Webfont Example. SIL has detailed information, see Using SIL Fonts on Web Pages.

If this document is not displaying correctly a PDF version is also provided in the documentation/pdf folder of the release package.
