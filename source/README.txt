NLCI Gujarati fonts
====================

This file describes the source files included with the NLCI Gujarati fonts.
This information should be distributed along with the NLCI Gujarati fonts and any derivative works.

This Font Software is licensed under the SIL Open Font License, Version 1.1.

master.sfd            - OpenType rules
other .sfd files      - glyph outline data for each font
gir_classes.xml       - class information for the attachment point database
gir_unicode.xml       - mapping from legacy to Unicode as used by ttfbuilder
GUJ Gir 2 Unicode CONV chart.xls - source for gir_unicode.xml
